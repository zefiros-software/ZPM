Premake5 Commands¶

These type of build command translate directly to premake5.

Example¶

//_build.json
[
    {
        "project": "Example",
        "do": [
            {"architecture": "x86_64"},
            {"endian": "Little" },
            {"links": [
                "mpi",
                "pthreads"
            ]}
        ]
    }
]
